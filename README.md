# devexp

Developer Experiments

#### Getting started
Requirements: mongodb, nodejs >= 0.12, forever.

##### Setup Prod:
```bash
npm install --production

// run
npm run forever:start
```

##### Setup Dev:
```bash
npm install

// run
npm run dev
```
